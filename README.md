# Desktop Icon Organizer（桌面图标整理工具）

A Windows desktop icon organization tool with scan, classify, preview, overlay, backup and apply workflow.

一款用于 Windows 的桌面图标整理工具，支持扫描、分类、预览、边框叠加、备份与一键应用。
